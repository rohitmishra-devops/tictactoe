# 🎮 Tic Tac Toe Game

A simple and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Two-player mode (X vs O)
- Winner detection
- Draw detection
- Reset Game button
- New Game button
- Responsive design
- Simple and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
tictactoe/
│── hello.html
│── style.css
│── app.js
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open `hello.html` in your browser.
3. Enjoy playing!
